题目 1022.矩阵乘法

Description

给出两个的矩阵，求他们的乘积

Input

第一行有两个正整数N1, M1 (0<N1, M1<=100)，
接下来有N1行，每行有M1个数字，描述第1个矩阵。
第N+2行有两个正整数N2, M2 (0<N2, M2<=100)
接下来有N2行，每行有M2个数字，描述第2个矩阵。
矩阵的每个元素在[-300, 300]之间，输入保证M1等于N2。

Output

输出N1行每行M2个数字表示结果矩阵。
每行中每个数字之间用一个空格隔开。

Sample Input

3 3
1 2 3
4 5 6
7 8 9
3 3
1 0 1
0 1 0
0 0 1

Sample Output

1 2 4
4 5 10
7 8 16

想法

直接用普通的矩阵乘法就过了，
时间复杂度 O()
